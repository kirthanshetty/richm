We recommend you to try the following steps to diagnose the issue:

Find out whether you can access any other internet sites. If you are sure the issue is specifically related to this site, then take a look at the error message.

If this message relates to a ‘network error message’, it could be that you are accessing our website from within a corporate network which is blocking access.

If some content is not displaying correctly, such as the ‘job offers’ or the ‘candidate overview’, this could be due to popup blockers in your web browser.

We would suggest you disable the popup blockers by changing your web browser settings. For example:

  • for Microsoft Internet Explorer 7: open browser, go to Tools, Pop Up Blocker, Turn off Popup blocker.
  • for Firefox 2.0: open browser, go to Tools menu, Options, click the Content icon located across the top of the window, click the Exceptions button located to the right of Block pop-up windows.

For security reasons, the system will perform an automatic logout after 30 minutes of inactivity.

If you are still experiencing technical problems, please email e.careers©richemont.com with a description and, ideally, a screenshot of the error you are experiencing.
